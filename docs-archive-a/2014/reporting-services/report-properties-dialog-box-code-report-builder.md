# <a name="report-properties-dialog-box-code-report-builder"></a>[コード] ([レポートのプロパティ] ダイアログ ボックス) (レポート ビルダー)
  **[レポートのプロパティ]** ダイアログ ボックスの **[コード]** を選択すると、レポート内の式によって使用されるカスタム コードを定義できます。
