# <a name="version-property--wmi-msreportserver_configurationsetting"></a>Version プロパティ (WMI MSReportServer_ConfigurationSetting)
  Major.Minor.Build.Revision 形式でレポート サーバーのバージョンを返します。 読み取り専用です。  
  
## <a name="syntax"></a>構文  
  
```vb  
Public Dim Version As String  
```  
  
```csharp  
public string Version;  
```  
  
## <a name="property-value"></a>プロパティ値  
 レポート サーバーのバージョンを表す `string`。
