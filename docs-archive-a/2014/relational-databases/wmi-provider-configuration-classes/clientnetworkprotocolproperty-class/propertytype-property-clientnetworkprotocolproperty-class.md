# <a name="propertytype-property-clientnetworkprotocolproperty-class"></a>PropertyType プロパティ (ClientNetworkProtocolProperty クラス)

## <a name="syntax"></a>構文  
  
```  
  
object  
.PropertyType [= value]  
```  
  
## <a name="parts"></a>指定項目  
 *object*
