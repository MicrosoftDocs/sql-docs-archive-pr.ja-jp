# <a name="complete---prepare-image"></a>[完了 - イメージの準備]
  [**完了**] ページを使用して、イメージの準備が完了したことを確認します。  
  
## <a name="options"></a>Options  
 このダイアログ ボックスには、この操作のセットアップ ログ ファイルへのリンクが含まれています。 イメージの準備を完了するには、[**閉じる**] をクリックします。
