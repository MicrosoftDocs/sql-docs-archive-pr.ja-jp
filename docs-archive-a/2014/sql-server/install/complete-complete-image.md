# <a name="complete---complete-image"></a>[完了 - イメージの完了]
  **[完了 - イメージの完了]** ページを使用して、準備済みインスタンスの構成が完了したことを確認します。  
  
## <a name="options"></a>Options  
 このダイアログ ボックスには、この操作のセットアップ ログ ファイルへのリンクが含まれています。 インストールを完了するには、 **[閉じる]** をクリックします。
